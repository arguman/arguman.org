User details
=======================

| Key             | Value                      |
| ----------------|----------------------------|
| URL             | /api/v1/users/`<username>`/ |
| Method          | GET                        |
| Content-Type    | application/json           |


##### Request

```bash
curl -X GET -H "Content-Type: application/json"
      http://arguman.org/api/v1/users/bahattincinic/
```

```json

  "id": 515,
  "username": "bahattincinic",

}
```
